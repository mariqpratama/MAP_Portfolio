A web-based information system for patient treatment data in Ciumbuleuit Health Center. This project is made as an Informatics final thesis in Parahyangan Catholic University.

### Build with
- HTML
- CSS (Bootstrap)
- JavaScript
- EJS
- PostgreSQL
